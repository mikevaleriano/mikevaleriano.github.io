Site name

The site's name is going to be Town's Weather and it's domain name will be townsweather.com which is still available on Internet.

Site Purpose

The main objective of townsweather.com is to provide a tourist guide in which the users can consult the weather of the city where they are, once the website knows their location, it will begin to show useful information and images about the best places to visit, eat or dance in the town.

The website will use location's coordinates to determine user's location so that he can see the rigth place's weather and google places API to provide information about places nearby.
